# 个人配置说明

这个 fork 的维护策略：

- `lua/lazyvim/`、`doc/`、`tests/` 等目录尽量保持上游 LazyVim 原样，方便以后同步官方更新。
- `user-config/` 是个人 Neovim 配置模板，可以复制到本机 Neovim 配置目录。
- `docs/` 是中文使用说明、Codex CLI 建议和调用图方案记录。

日常更新建议：

```sh
git fetch upstream main
git merge upstream/main
git push origin main
```

如果你只想更新个人配置，通常只改：

```text
user-config/lua/config/
user-config/lua/plugins/
docs/
```
